# 多角色自动跳转系统测试用例文档

## 1. 测试环境
- **前端**: Vue 3 + Vue Router
- **后端**: Spring Boot + MyBatis Plus
- **数据库**: MySQL
- **测试账号**:
    - 学生: 注册新账号 或 使用现有学生账号
    - 教师: `teacher` / `123456` (系统自动初始化)
    - 管理员: `admin` / `123456` (假设存在)

## 2. 功能测试用例

### 2.1 登录跳转测试

| TC-003 | 验证管理员登录跳转 | 未登录 | 1. 访问 `/login`<br>2. 输入管理员账号密码<br>3. 点击登录 | 系统提示登录成功，自动跳转至 `/admin-dashboard` | 待验证 |

### 2.3 会话状态保持与自动重定向

| 用例ID | 测试目的 | 前置条件 | 操作步骤 | 预期结果 | 实际结果 |
